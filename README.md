# Placement Prediction and Skill Improvement Dashboard

## Problem Statement
University placement cells often struggle to systematically identify students who are "at-risk" of remaining unplaced. Without deep intervention strategies, generic advice fails. This project aims to analyze a holistic student profile (CGPA, Coding, Communication, Internships, etc.), predict their placement probability, and programmatically generate actionable, skill-specific recommendations.

## Dataset Description
The dataset contains detailed engineering student profiles:
- **Academics**: CGPA, Backlogs, Branch
- **Skills**: Aptitude_Score, Communication_Score, Coding_Score
- **Experience**: Internship_Count, Project_Count
- **Targets**: Placement_Status (Placed/Not Placed), Company_Type (Product/Service), Package_Offered

## Tools Used
- **SQL Server / SQLite**: Data ingestion, normalization, and outlier standardization. 
- **Python**: 
  - `Pandas`, `Matplotlib`, `Seaborn` for Exploratory Data Analysis.
  - `Scikit-Learn` for Logistic Regression and Decision Tree classification architectures.
- **Power BI**: Interactive KPI deployment and dark-theme data visualization.

## Steps to Run the Project
1. **Initialize Environment**: Ensure Python 3.8+ is installed. Run `pip install pandas numpy scikit-learn matplotlib seaborn`.
2. **Generate / Analyze Dataset**: 
   - Execute `python/generate_dataset.py` to compile the baseline CSV (if necessary).
3. **Train Models**:
   - Execute `python/eda_and_models.py` to auto-generate correlation matrices in `reports/` and serialize the best-fit classifier into `model.pkl`.
4. **Trigger Recommendations**:
   - Run `python/skill_engine.py` to output the probability shifting logic row-by-row.
5. **Dashboard Visualization**:
   - Open Power BI. Import `dataset/placement_dataset.csv`. Follow instructions in `dashboard/dashboard_layout.md` for designing the frontend visual logic.

## Business Insights
- **Coding Threshold**: In CS and IT branches, coding scores below 70 natively eliminate product company possibilities entirely; intensive bootcamps are required.
- **Internship Viability**: A single internship significantly multiplies probability odds regardless of a lower CGPA.
- **Communication Floor**: Service tier companies utilize strict communication baseline metrics (60+) to pass HR screening.
